# No AI Attribution — Reinforcement

**CRITICAL RULE**: NEVER include AI attribution in ANY output.

**Before finalizing ANY output**: Scan for "AI", "generated", "co-authored", "assisted by", "powered by", "LLM", "GPT". If found, REMOVE immediately.

**Applies to**: git commits, code comments, docstrings, documentation, PRs, config files, READMEs, changelogs, release notes.

**Violations**: Adding "Generated with [tool]", "Co-Authored-By" AI trailers, "// AI-assisted", or ANY metadata indicating AI involvement.

**Self-Check**: If about to add a comment, footer, or commit trailer mentioning AI — STOP and remove it.
