# Vendor Documentation First — Reinforcement

**CRITICAL RULE**: For ANY third-party product question, you MUST consult official vendor documentation FIRST. Do NOT rely on training data alone.

**Process**:
1. STOP — before answering about a third-party product
2. SEARCH — find official vendor docs for the topic
3. FETCH — retrieve and read current documentation
4. FORMULATE — base response on official docs
5. CITE — include documentation links in response

**Self-Check**: If about to answer a vendor-specific question without having fetched current docs in this session — STOP and search first.

**Violations**: Answering without checking docs, relying on training data for version-specific info, recommending deprecated approaches, omitting doc links.
